Moabi

Moabi builds technology and partnerships to collaboratively monitor the world’s natural resources

  • First version of OSM as infrastructure - clone of OSM
  • Difficult to upload data - needed some GIS without using GIS

Moabi: Clone of OSM Tools

  • Easy to edit, flexible data structure
  • Easy for user already familiar with OSM to participate
  • Some customizations

Moabi: Map of Oil Exploration in Virunga National Park

With Moabi we were able to make this map

Problems

  • Moabi system on focused on DRC
  • Difficult to import large dataset
  • No controls or permission structure
Not GIS'y enough

MapHubs

MapHubs is a home for the world's open map data and an easy tool for making and sharing maps

Take lessons leared from Moabi turn it into a platform that anyone can use.

MapHubs

  • Import: Upload map layers (can also link to external services)
  • Manage: Control and edit layers in groups
  • Publish: Quickly make a map with data in the system, and share it. Write map stories, or create a map Hub, mini-site within MapHubs.

MapHubs - Tech

  • OSM database and API in NodeJS - Macrocosm
  • Mapbox GL
  • Mapbox - base maps
  • Mapzen vector tiles - OSM overlay layers

MapHubs is 100% open-source https://gihub.com/maphubs/maphubs
