OSM is like Wikipedia?

  • Requires a concensus
  • Anyone can edit
  • Data must be releasable under the open license
  • One Truth
  • Useful Information
  • Completely replacing a page with your "better" version?

What doesn't belong in Wikipedia?

  • List of all fictional vice presidents
  • List of lists of lists
  • My movie script idea...

What doesn't belong in OSM?

  • Pokémon Go - PokéStops?
  • My favorite places with free wifi
  • Land I own
Last one is serious, property rights whether ownership or permission from the government for mining/logging

Moabi

  • First version of OSM as infrastructure - clone of OSM
  • Difficult to upload data - needed some GIS without using GIS

MapHubs

MapHubs is a home for the world's open map data and an easy tool for making and sharing maps

  • Quickly map a map, overlaying evironmental data, data from OSM, or data your create
  • Tell your story: write story on MapHubs or embed a map in Medium or your website
  • Our customers = more familiar with OSM, too busy to also learn GIS, but we need some GIS features

MapHubs Tech

  • OSM database and API in NodeJS - Macrocosm
  • Mapbox GL
  • Mapbox - base maps
  • Mapzen vector tiles - OSM overlay layers

MapHubs is 100% open-source https://gihub.com/maphubs/maphubs
